# XMCMC website

A simple, static, Wikipedia-inspired website for the XMCMC GitHub organization/project.

## Files

- `index.html` - main page
- `docs/` - documentation page
- `examples/` - examples page
- `reference/` - API reference placeholder
- `community/` - contributing and community page
- `changelog/` - release notes page
- `assets/style.css` - all styling
- `assets/xmcmc-mark.svg` and `assets/xmcmc-logo.png` - logo assets

## GitHub Pages deployment

For an organization site, create a repository named `XMCMC.github.io` under the `XMCMC` organization, then commit these files to the repository root. In GitHub, open **Settings > Pages** and publish from the branch/root folder if it is not enabled automatically.

For a project site, commit these files to a repository such as `XMCMC/XMCMC-site` and configure **Settings > Pages** to publish from the root or `/docs` folder.

## Editing

This is plain HTML and CSS. Edit the pages directly in GitHub or locally, commit, and push.
