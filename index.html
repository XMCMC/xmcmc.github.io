Overview

XMCMC is intended to provide a small, composable interface for sampling from user-defined target distributions. The project can document algorithms, diagnostics, examples, and implementation details here without requiring a full documentation build system.

General purpose

Use XMCMC as a shared home for MCMC kernels, target definitions, diagnostics, and examples.

Research friendly

Keep mathematical notes, examples, and implementation details close to the source code.

Simple publishing

This site is plain HTML/CSS, so it can be served directly with GitHub Pages.

Wiki-inspired

The layout borrows the plain reading experience, sidebar navigation, infobox, and table-of-contents feel of Wikipedia.
