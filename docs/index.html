    Core concepts

    TermMeaning
    Target distributionThe probability distribution to sample from, usually represented by a log density.
    KernelA transition rule that proposes or updates the current state.
    ChainThe sampled sequence of states.
    DiagnosticsChecks for convergence, mixing, and effective sample size.
