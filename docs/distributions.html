XMCMC Distribution Reference

This page lists the distributions currently exposed by the XMCMC model translator/header. Distribution names follow the XMCMC convention of using d-prefixed names such as dnorm, dbern, dmvn, and dinvwish.

Two ways to use a distribution

1. Sampling-statement syntax

Use this when you want XMCMC to translate the distribution statement into an addition to the current block target.

theta ~ dnorm(0, 1);
y(i) ~ dnorm(mu(i), sigma);
z(i) ~ dbern(p(i));

For vector or matrix-like contiguous objects, use a range on the left side:

vec utem(2);
vec zero2(2);
mat Sigma(2,2);

utem(1:2) ~ dmvn(zero2, Sigma);

2. Manual target-accumulator syntax

Use this when you want to write the log-density contribution explicitly. Blocks automatically have an accumulator named target, so users do not need to declare or return it.

target += dnorm_lpdf(theta, 0, 1);
target += normal_lpdf(theta, 0, 1);
target += dmvn_lpdf(utem, zero2, Sigma, 2);
Note: The sampling-statement syntax generally omits the left-hand variable from the argument list. The manual target += ..._lpdf(...) syntax includes the sampled value as the first argument.

Aliases

The translator also accepts some non-d aliases, mainly for compatibility: normal, uniform, exponential, gamma, invgamma, beta, cauchy, student_t, chisq, lognormal, logistic, laplace, weibull, pareto, halfnorm, halfcauchy, bernoulli, binomial, poisson, negbinomial, mvn, multi_normal, multivariate_normal, bvn, bivariate_normal, wishart, invwishart, inv_wishart, and lkjcorr2.

Example: covariance prior with a local matrix

block u(j,l) {
  vec utem(2);
  vec zero2(2);
  mat Sigma_u(2,2);

  double tau0 = exp(logtau0(1));
  double tau1 = exp(logtau1(1));
  double rho = tanh(rho_raw(1));

  Sigma_u(1,1) = tau0 * tau0;
  Sigma_u(2,2) = tau1 * tau1;
  Sigma_u(1,2) = rho * tau0 * tau1;
  Sigma_u(2,1) = Sigma_u(1,2);

  zero2(1) = 0;
  zero2(2) = 0;

  utem(1) = u(j,1);
  utem(2) = u(j,2);

  utem(1:2) ~ dmvn(zero2, Sigma_u);
}

Example: manual target syntax

block logtau0(1) {
  vec utem(2);
  vec zero2(2);
  mat Sigma_u(2,2);

  /* fill Sigma_u here */

  target += dnorm_lpdf(logtau0(1), 0, 2);

  for (j = 1:m) {
    utem(1) = u(j,1);
    utem(2) = u(j,2);
    target += dmvn_lpdf(utem, zero2, Sigma_u, 2);
  }
}
